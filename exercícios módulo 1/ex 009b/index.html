Trabalhando com listas

Listas ordenadas

São listas que seguem um padrão de ordem. (OL) (OrdemList)

o final /li é opicional

  1. Acordar
  2. Tomar café
  3. Ir para academia
  4. Comprar frutas

Posso alterar o tipo da ordem usando a tag type dentro da tag ol, e também posso usar a tag start dentro da type, para seguir a sequência. Caso seja letra, a regra muda um pouco, eu uso a mesma tag, porém invés de colcoar a letra, eu sigo o número, ex type c / start 3.

Listas não ordenadas

São listas que não são numeradas. (UL) (UnorderList)

  • Pão
  • Queijo
  • Patê
  • Banana

Posso alterar o símbolo da listagem, usando a tag type dentro da ul, porém não fica aparece a tag, só digitar a tag type e = .

  • disc (disco preto)
  • circle (bola aberta)
  • Square (quadrado)
