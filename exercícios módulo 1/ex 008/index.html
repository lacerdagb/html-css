Negrito / Destaque

Nessa frase eu estou usando a tag B, que é errado, não faz sentido, pinta as letras visualmente, mas não tem sentido nela, e é errado usar

Porém eu posso usar o comando <strong> para dar sentido e forma para a palavra ou frase, que usarei nesse instante a forma correta de se usar

Itálico

O comando italico funciona o comando <i> que usarei nesssa frase porém assim como o negrito, ela está errada

O termo certo é <em> que vem de ênfase que usarei nesse instante

Dica extra: Eu posso usar o comando Ctrl+Shift+P para usar atalho de tags
